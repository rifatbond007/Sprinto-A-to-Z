## Description

Tracks third-party vendor security postures and maintains vendor compliance documentation.

- **Vendor inventory**: Centralized registry of all vendors
- **Security questionnaires**: Automated vendor assessment workflows
- **Breach monitoring**: Real-time alerts for vendor security incidents
- **Risk scoring**: Automated vendor risk ratings
- **Evidence storage**: Stores vendor SOC 2 reports, policies, and assessments

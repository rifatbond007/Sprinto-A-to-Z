## Description

Intelligently maps controls, policies, and evidence across multiple frameworks to eliminate duplicate work.

- **Control mapping**: Links shared controls across SOC 2, ISO 27001, HIPAA, etc.
- **Evidence reuse**: Single piece of evidence satisfies multiple controls
- **Gap analysis**: Identifies missing controls for each framework
- **Framework comparison**: Highlights differences between standards

---

## Example Mapping

| Control | SOC 2 | ISO 27001 | HIPAA |

| Encryption | CC6.7 | A.10, A.12 | §164.312(e) |
| Logging | CC7.2 | A.12 | §164.312(b) |

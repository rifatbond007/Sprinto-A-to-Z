---

## Description

Runs always-on checks against security controls and alerts when issues arise.

- **Real-time checks**: Monitors MFA, password policies, logging, encryption settings
- **Custom checks**: Define specific conditions to monitor
- **Alerting**: Sends notifications via Slack, email, or webhooks
- **Trend dashboards**: Visualizes compliance health over time
- **Remediation guidance**: Provides step-by-step fix instructions

---

## Monitored Controls Include

- Multi-factor authentication status
- Password policy compliance
- Cloud storage encryption
- Logging and monitoring enablement
- Security group configurations
- Device compliance status

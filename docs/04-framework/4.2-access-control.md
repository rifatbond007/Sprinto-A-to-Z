## Description

Manages user identities, permissions, and access policies across all integrated systems.

- **Identity integration**: Connects with Okta, Google Workspace, Azure AD
- **MFA enforcement**: Monitors and validates multi-factor authentication status
- **Access reviews**: Automates periodic access certification campaigns
- **Least privilege**: Tracks and enforces permission best practices
- **User lifecycle**: Manages onboarding, role changes, and offboarding

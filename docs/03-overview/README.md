## What Sprinto Does

Sprinto provides a unified platform to:

1. **Automate evidence collection** from 200+ integrations
2. **Monitor security controls** in real-time
3. **Manage policies** with auditor-approved templates
4. **Assess and track vendor risks**
5. **Generate audit-ready reports** instantly
6. **Maintain continuous compliance** across multiple frameworks

---

## Supported Frameworks

Sprinto supports **40+ security and privacy frameworks**:

| Framework | Use Case | Certification Type |
|-----------|----------|-------------------|
| **SOC 2** | Service organization control reporting | Type I & Type II |
| **ISO 27001** | Information security management (ISMS) | Certification |
| **HIPAA** | Healthcare data protection | Compliance |
| **GDPR** | EU data privacy | Compliance |
| **PCI DSS** | Payment card security | Compliance |
| **NIST** | Cybersecurity framework | Compliance |
| **FedRAMP** | US government cloud security | Authorization |
| **CMMC 2.0** | Defense contractor cybersecurity | Certification |
